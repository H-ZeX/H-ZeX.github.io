  1. 1. Condition codes

Condition codes

  • leaq: set none
  • integer arithmetic operations: cause the codes to be set.
  • logical operations:
    • xor: carry and overfolow flags are set to zero
    • shift operations: set carry to the last bit shifted out, overflow flags set to zero
    • INC and DEC instructions set the overflow and zero flags but leave the carry flags unchanged
